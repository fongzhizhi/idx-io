# 使用案例说明

> 案例代码目录：`examples/`
>
> 案例输出目录：`outputs/`
>
> 特别说明：每个案例都会展示两种不同的建模方式：传统建模和简化建模，对应输出两个`idx`文件。命名方式：
>
> - 传统建模：`{exampleFileName}.trad.idx`
> - 简化建模：`{exampleFileName}.simp.idx`

## 案例1：简单板子

+ 文件名：`01.simple-board.ts`
+ 案例目的：演示`IDXExporter`基本用法，展示`IDX`基本结构。
+ 案例说明：一个简单的矩形板，厚度为1.6mm，无其他项目（Item）。

## 案例2：层和层堆叠

- 文件名：`02.layer-and-layer-stackup.ts`
- 案例目录：演示层和层堆叠的建模方式，展示层和层堆叠在`IDX`中的结构。
- 案例说明：一个四层板（底层、内层1、内层2、顶层），层直接有介电层，板子需要关联层堆叠，无其他项目。

## 案例3：元件

- 文件名：`03.components.ts`
- 案例目的：演示元件的建模方式，展示元件以及元件相关数据在`IDX`中的结构。
- 案例说明：继承案例2，四层板，项目有：
  1. 3D模型：R0402。
  2. 3D模型：R0403。
  3. 封装R0603：一个简单的矩形轮廓和两个引脚信息组成。绑定了3D模型R0402。
  4. 封装M3：一个简单的圆形轮廓，无引脚信息。
  5. 元件R1：电气元件，电阻，封装为R0603，顶层，坐标为(5, 20)，旋转角度为45度，继承了封装的3D模型R0402。
  6. 元件R2：电气元件，电阻，封装同R1为R0603，底层，坐标为(15, 15)，旋转角度为180度，绑定了3D模型R0403。
  7. 元件TP1：机械元件，螺丝，封装为M3，顶层，坐标为(3, 3)，旋转角度为0度。

## 案例4：过孔

- 文件名：`04.vias.ts`
- 案例目的：演示过孔和盲埋孔的建模方式，展示过孔和盲埋孔相关数据在`IDX`中的结构。
- 案例说明：继承案例2，四层板，项目有：
  1. 通孔（过孔）：贯穿整个板子。
  2. 盲孔：内层1到顶层
  3. 埋孔：内层1到内层2

## 案例5：禁止区和保留区域

+ 文件名：`05.keepout-and-keepin.ts`
+ 案例目的：演示禁止区和保留区域的建模方式，展示禁止区和保留区相关数据在`IDX`中的结构。
+ 案例说明：继承案例2，四层板，项目有：
  1. 矩形禁止区：禁止元件
  2. 多边形保留区：保留元件

## 案例6：完整多层板

- 文件名：`06.complete-multi-board.ts`
- 案例目录：完整演示各项目的建模方式，展示相对完整的`IDX`结构。
- 案例说明：整合案例2-5，输出一个有层堆叠包含各种项目的四层板。
